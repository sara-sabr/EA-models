Lite-2021-IITB-10 SDIV-MVP2 for CSLP ()
Project description per Lite Proposal: The solution scope is to augment the ECAS solution by integrating the SDIV functionality to provide digital identity validation capability. This form of validation provides an additional option and improves user experience by complimenting, not replacing, the current validation mechanisms. Existing NSLSC users will have the choice to gain access using the now present security questions and answers capability, or through the new SDIV identity validation functionality. The SDIV validation capability will be available to existing users for substitutions only. Once the substitution is successful and the profile is created, students may resume access to the NSLSC portal for online services.
