Desired Outcomes ()
2.3 Desired Outcomes • The new conditions to the system are added and can be selected when an inspection is created and initiated; • New letter(s)/annex(es) are added and the existing templates are updated in the Corporate Correspondence Tool (CCT) with the new conditions; • Updates to the solution are made in order to continue the use of CCT to produce letters/annexes directly from the case. • Additional fields are added to the solution to request additional information/documents required to verify employers’ compliance with the new conditions. • The Training site is updated with the changes to facilitate training prior to deploying changes to production. • The program is ready to start inspecting employers against the new conditions imposed as of September 26th, 2022.
