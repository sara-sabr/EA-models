GCcollab File Sharing (Application Service) ()
Uploading a video Uploading a video file on GCcollab uses the exact same steps as uploading a regular file, however it is important to note that the size of the video to be uploaded should not exceed 1GB. While GCcollab does not have video player functionality, users can upload video file types. Accepted file formats:: txt, jpg, jpeg, png, bmp, gif, pdf, doc, docx, xls, xlsx, ppt, pptx, odt, ods, odp, accdb, mdb, m4a, mp4, grd, map, rar, gdb, dwg, zip, mp3, ppsx, mid, mov, xlsm, ai, xd, svg, indd, vsd, vsdx, mpp, mppx, potx, dotx, .dotx
