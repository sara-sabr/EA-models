YAML (YAML Ain’t Markup Language) (Business Object) ()
Name: YAML (a recursive acronym for “YAML Ain’t Markup Language”) Description: Functionally a superset of JSON that is intended to serve similar applications to XML, but with a more human-friendly syntax. Its syntax also differs from that of JSON. It is commonly used for configuration files and in applications where data is being stored or transmitted. Source/URL: Refer to: https://yaml.org.
