Turing eXtender Language (TXL) (Business Object) ()
Name: Turing eXtender Language (TXL) Description: A programming language designed to support computer software analysis and source transformation tasks. A TXL program specifies the structure of the input to be transformed using an extended form of BNF, and gives rules for rewriting the input to produce output. It enables structural analysis and transformation of formal notations, and has been used in a wide range of applications, including database schema translation.
