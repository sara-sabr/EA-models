Hybrid Data The structured component of hybrid data sets is often there to provide metadata. Word processor files often include metadata, such as the name of the author. The HTML standard defines the <meta> tag to include metadata in web pages, and also allows linked data, which is structured data that is incorporated in web pages so that they become more useful through semantic queries. JavaScript Object Notation for Linked Data (JSON-LD)21 is a standard method of encoding linked data using JSON. Refer to: https://www.w3.org/TR/json-ld11/.
