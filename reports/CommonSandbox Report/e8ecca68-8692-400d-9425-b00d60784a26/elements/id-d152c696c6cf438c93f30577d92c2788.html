Guideline A.3.3: (For the authoring tool user interface) Help authors avoid flashing that could cause seizures. [Implementing A.3.3] Rationale: Flashing can cause seizures in authors with photosensitive seizure disorder. A.3.3.1 Static View Option: If an editing-view can play visual time-based content, then playing is not necessarily automatic upon loading the content and playing can be paused.
