Application architecture click here for Application Architecture guide https://wiki.gccollab.ca/GC_Enterprise_Architecture/Framework/ApplicationGuide Application architecture practices must evolve significantly for the successful implementation of the GC Enterprise Ecosystem Target Architecture. Transitioning from legacy systems based on monolithic architectures to architectures that oriented around business services and based on re‑useable components implementing business capabilities, is a major shift. Interoperability becomes a key element, and the number of stakeholders that must be considered increases.
