EIRA Solution Building Block ()
DEFINITION: EIRA Solution Building Block ABB is an Artifact being a concrete component of an intermediate level of granularity, that it implements one or more EIRA Architecture Building Blocks of an European Public Service, formulated as an agreed normative statement in design terms on a legal, organisational, semantic, or technical attribute of an European Public Service. On the technical view, a Solution Building Block is a specific software component that it might be either procured or developed of a To-Be Interoperable European Solution or that it is integrated in an As-Is Interoperable European Solution Source: Business Architecture (Whelan, J.; Meaden, G. (2012). Business Architecture: A practical guide. Ashgate.) EXAMPLES: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): A network is a building block that can be described through complementary artifacts and then put to use to realize solutions for the enterprise.
