UX Management Component ()
DEFINITION: UX Management Component ABB is Application Component managing user experience design-related activities inside an organization to create growth and good management practices. Typical UX management activities are to define an organization’s UX design language and strategy and manage the work processes around UX design. Source: Interaction design (https://www.interaction-design.org/literature/topics/ux-management)
