Accountability ()
DEFINITION: Accountability ABB is a Principle enabling the accountability of users and of public administrations. Transparency, liability, controllability, responsibility, and responsiveness are defined as distinct dimensions of accountability, providing an improvement on the current state of conceptual fuzziness. The typology provides a vocabulary for the core argument, that conflicting expectations borne of disparate conceptions of accountability undermine organizational effectiveness. Source: (Williams, Reyes (2006) Leadership accountability in a globalizing world.
