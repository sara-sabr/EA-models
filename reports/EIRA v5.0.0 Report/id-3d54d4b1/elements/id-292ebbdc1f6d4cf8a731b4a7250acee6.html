DEFINITION: Blockchain Component ABB is an Application Component consisting of various components like peer network, smart contract, membership, events, ledger, system integration, wallet, and system management. Several actors such as developer, user, architect, regulator, operator, membership services, etc. are present in a blockchain that provides a solution to a blockchain-based business network. Each actor and component present in a blockchain plays a major role to make the network more secure and reliable.
